# YouTube Bulk URL Scraper

Extension Chrome pour extraire les URLs de vidéos YouTube et les transcripts.

## Fonctionnalités

### Extraction d'URLs (pages de recherche)
- Extraction automatique via auto-scroll (50 vidéos)
- Tri par date (plus récentes en premier)
- Filtre par ancienneté (1 semaine, 1 mois, 6 mois, 1 an)
- Sélection et copie des URLs

### Extraction de transcript (pages vidéo)
- Récupération automatique des sous-titres
- Préférence pour le français, fallback sur la langue disponible
- Toggle timestamps (avec ou sans)
- Copie du transcript

## Installation

1. Aller sur `chrome://extensions/`
2. Activer le "Mode développeur"
3. Cliquer sur "Charger l'extension non empaquetée"
4. Sélectionner ce dossier

## Utilisation

### Mode Scraper (recherche YouTube)
1. Aller sur une page de recherche YouTube
2. Cliquer sur l'icône de l'extension
3. Cliquer sur "Lancer l'extraction"
4. Filtrer par date si nécessaire
5. Copier les URLs sélectionnées

### Mode Transcript (vidéo YouTube)
1. Aller sur une vidéo YouTube
2. Cliquer sur l'icône de l'extension
3. Cliquer sur "Extraire le transcript"
4. Activer/désactiver les timestamps
5. Copier le transcript
